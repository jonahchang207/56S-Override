# Vortex: Autonomous Motion & Odometry for VEX V5 Robotics
Welcome to the documentation home for **Vortex**, a professional C++ motion-control and pose estimation library built for VEX V5 Competition Robots (Team **56S-Override**).

Developed as an **AP Computer Science A Final Project** by AP CS student **Jonah Chang**, this repository showcases the intersection of advanced coordinate mathematics, software engineering best practices, and real-world embedded systems.
